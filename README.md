# Dotfiles

# Prerequisites

- Ubuntu 16.04+
- GNU Stow

# Get Started

```shell
sudo apt install stow
stow mybashrc ideavim tmux nvim
echo '[ -f ~/.mybashrc ] && source ~/.mybashrc' >> ~/.bashrc
```

More details are included in separated pages. Check below:
